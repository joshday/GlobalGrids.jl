# GlobalGrids

[![Build Status](https://github.com/joshday/GlobalGrids.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/joshday/GlobalGrids.jl/actions/workflows/CI.yml?query=branch%3Amain)
[![Build and Deploy Docs](https://github.com/joshday/GlobalGrids.jl/actions/workflows/Documentation.yml/badge.svg)](https://github.com/joshday/GlobalGrids.jl/actions/workflows/Documentation.yml)


This package contains:

- Implementations of different [Discrete Global Grid (DGG) systems](https://en.wikipedia.org/wiki/Discrete_global_grid) in Julia.
- Abstractions for working with a DGG.
- Utilities for creating DGGs based on an icosahedron.
